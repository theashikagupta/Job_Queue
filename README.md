# Job_Queue
AI-powered job search assistant that matches jobs with your resume, saves relevant opportunities into a queue, and helps track applications.

## Project Structure

- `backend/` - Node.js, Express, MongoDB, resume parsing, job search, auth, and application routes.
- `frontend/` - React + Vite dashboard.
- `.env` can stay at the project root; the backend reads it from there.

## Run Commands

Start the backend from the project root:

```bash
npm run dev
```

Start the React frontend from the project root:

```bash
npm run frontend
```
